# Fruit and Vegetable Detection using YOLOv8

This project detects fruits and vegetables using YOLOv8 object detection.

## Tech Used
- Python
- YOLOv8
- OpenCV
- Google Colab

## Classes Detected
- Apple
- Banana
- Blackberries
- Chilli
- Grapes
- Lemon
- Raspberry
- Tomato
